# Bio-Grid 规则设定详解

Bio-Grid 是一个基于网格的生态热力学模拟器，主要使用能量（Energy）与生物量（Biomass）两项数值来衡量生命的兴衰。这里详细说明了每一个网格（细胞）的生存、繁衍与死亡规则。

## 1. 核心属性与基础参数

每个格子（细胞）包含以下状态：
- **Type**：类型（0 = 空地，1 = 植物，3 = 墙体）。
- **Biomass**：生物量（0 到 1 之间）。当生物量归零时，细胞彻底死亡变为空地。
- **Energy**：能量储备（0 到 36 之间）。维持生命、抵抗密度压力、繁衍后代均需要消耗能量。
- **Gene**：基因值（0 到 1 之间）。影响细胞的光合作用效率、基础代谢率以及视觉颜色。

### 全局循环参数与时间尺度（重要）
- **Tick 与昼夜更替**：模拟器每次 Tick 步进时，内部时间（Time）增加 `0.05`。光照更替速度（Speed）默认值为 `0.014`。
- **漫长的昼夜**：由于每次 Tick 正弦相位仅增加 `0.0007`（0.05 * 0.014），一个完整的半周期（纯白昼或纯黑夜，即 `π` 的相位跨度）长达约 **4488 个 Tick**。
- **Sunlight**：当前全局光照（0 到 1，随昼夜周期波动 `max(0, sin(Time * Speed))`）。
- **maxEnergy**：每个细胞能够储存的能量上限（现为 36）。超出部分的能量会被舍弃。
  - **尺度说明**：Energy 是模型内部数值标尺，不直接等同真实生态学单位。UI 中建议以归一化值（`Energy / maxEnergy`）观察状态变化。

## 2. 能量摄入与消耗（代谢）

每过 1 个 Tick，每个植物细胞都要计算其能量的收入与支出：

1. **光合作用收入（Income）**
   收入与当前的阳光强度和细胞自身的基因相关。高基因（激进）的细胞光合作用效率远超低基因（保守）细胞。
   - `Income = Sunlight * (0.02 + Gene * 0.05)`
   - *（注：目前版本暂时移除了地形肥力设定，所有格子均能公平获取光照。）*

2. **基础代谢支出（Cost）**
   维持生命的基础开销，基因越激进，代谢支出越高（呈平方级增长）。
   - `Cost = 0.0012 + (Gene * Gene * 0.0056)`

3. **能量扩散（Diffusion）**
   能量扩散是“守恒的局部流动”：每个格子会把自身能量（负值视为 0）按固定比例分成两部分——一部分留给自己，另一部分平均分给可扩散的邻居（墙体阻断）。
   - 扩散只在非负能量范围内发生：扩散计算中的能量视为 `max(0, Energy)`（负能量不会向外扩散）。
   - 留存比例：`keepFrac = diffuseSelf / (diffuseSelf + diffuseNeighbor)`
   - 外流比例：`outFrac = diffuseNeighbor / (diffuseSelf + diffuseNeighbor)`
   - 每个 Tick：`keep = max(0, SelfEnergy) * keepFrac`，`out = max(0, SelfEnergy) * outFrac`
   - `out` 会平均分配给所有“活体植物”邻居（`Type = PLANT` 且 `Biomass > 0`）；空地与墙体不会接收能量。
   - 若周围不存在可接收扩散的存活邻居，则 `out` 不会丢失，而是回流到自身（即全部留存）。

## 3. 密度压力与能量惩罚（拥挤与孤独机制）

Bio-Grid 采用**能量耗尽才凋亡**的抗压机制。不论是过度拥挤还是过度孤独，都会增加能量消耗。

1. **局部拥挤（微观）**
如果一个细胞周围 8 个邻居中，“活体植物”的数量**大于 5 个**（即有 6 到 8 个活体植物邻居时），该细胞会受到局部拥挤惩罚。由于自然界中的资源竞争（如遮光、抢水）往往是非线性的，因此超出 5 个后的密度惩罚呈**平方级**增长。
- 惩罚计算：`LocalCrowd = Max(0, 邻居数 - 5)`
- 局部能量流失：`(LocalCrowd * LocalCrowd) * 0.0034`
  - 6 个邻居（超1）：流失 `1 * 0.0034 = 0.0034`（可感知的额外压力）
  - 7 个邻居（超2）：流失 `4 * 0.0034 = 0.0136`（强压）
  - 8 个邻居（超3）：流失 `9 * 0.0034 = 0.0306`（致死级压力）

2. **孤独惩罚（微观）**
如果一个细胞周围 8 个邻居中，“活体植物”的数量**小于 2 个**（即只有 0 或 1 个活体植物邻居时），该细胞会因为缺乏群落的微气候保护而受到孤独惩罚。
- 孤独能量流失：`0.014`

其中“活体植物”在实现中指 `Type = PLANT` 且 `Biomass > 0`。

> **结论**：高密度或孤独都不会导致细胞立刻“爆体而亡”。而是会让细胞的能量消耗增加。只有当这些惩罚彻底把细胞的能量（Energy）抽干后，才会触发下文的凋亡逻辑。

## 4. 生长与凋亡（Biomass 结算）

在结算完所有能量的收入、基础开销与密度惩罚后，会得出细胞最终的剩余能量。

- **生长**：如果最终 `Energy > 0`，细胞处于顺境，其生物量（Biomass）会增加 `0.005`，最高不超过 `1.0`。
- **凋亡**：如果最终 `Energy <= 0`，细胞不仅能量归零，还会开始消耗“肉体”来抵债，其生物量（Biomass）会扣除 `0.002`（这给细胞留出了等待日出或熬过干旱的缓冲时间，而不是瞬间暴毙）。
- **死亡判定**：如果由于持续的能量枯竭，导致 `Biomass <= 0`，则该细胞彻底死亡，状态变为 `EMPTY`，基因与能量清零。

## 5. 繁衍与突变（双亲繁殖）

想要繁衍必须同时满足五个苛刻的条件：
1. **成熟度**：自身的生物量（Biomass）必须 > 0.92。
2. **能量储备**：自身的能量（Energy）必须 > 14。
3. **周围有空地**：8 邻域内至少有一个空地（EMPTY）。
4. **不拥挤**：自身的 8 邻域内植物邻居数量必须 <= 5。
5. **有合格的伴侣**：目标空地的周围，除了自身之外，必须还有至少另一个植物邻居同样满足上述的“成熟度”与“能量储备”条件。

如果满足以上条件，细胞会进行分裂：
- **产生后代**：随机选中满足上述条件的一个空地，生成新的细胞（也就是说，这个空地是由两个合格的母体共同孕育的）。
- **初始数值**：后代诞生时的初始生物量为 0.32。
- **能量平分**：两个母体将当前能量的四分之一（25%）分给后代，自己保留剩下四分之三。
- **基因突变**：后代会随机继承两个母体其中之一的基因，但会带有 `±0.04` 的随机突变波动。

## 6. 特殊网格：墙体（WALL）
- 墙体无法生长、死亡或繁衍。
- 墙体能量永久为 0。
- 墙体绝对**不参与**邻居的能量扩散与平均值计算（阻断能量流动）。
- 玩家可用鼠标中键手动绘制墙体，从而人为构造出“生态隔离舱”。
