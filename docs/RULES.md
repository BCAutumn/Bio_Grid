# Bio-Grid 规则设定详解

Bio-Grid 是一个基于网格的生态热力学模拟器，主要使用能量（Energy）与生物量（Biomass）两项数值来衡量生命的兴衰。这里详细说明了每一个网格（细胞）的生存、繁衍与死亡规则。

## 1. 核心属性与基础参数

每个格子（细胞）包含以下状态：
- **Type**：类型（0 = 空地，1 = 植物，3 = 墙体）。
- **Biomass**：生物量（0 到 1 之间）。当生物量归零时，细胞彻底死亡变为空地。
- **Energy**：能量储备（0 到 40 之间）。维持生命、抵抗密度压力、繁衍后代均需要消耗能量。
- **Gene**：基因值（0 到 1 之间）。影响细胞的光合作用效率、基础代谢率以及视觉颜色。

### 全局循环参数与时间尺度（重要）
- **Tick 与昼夜更替**：模拟器每次 Tick 步进时，内部时间（Time）增加 `0.05`。光照更替速度（Speed）默认值为 `0.014`。
- **漫长的昼夜**：由于每次 Tick 正弦相位仅增加 `0.0007`（0.05 * 0.014），一个完整的半周期（纯白昼或纯黑夜，即 `π` 的相位跨度）长达约 **4488 个 Tick**。
- **Sunlight**：当前全局光照（0 到 1，随昼夜周期波动 `max(0, sin(Time * Speed))`）。
- **maxEnergy**：每个细胞能够储存的能量上限（现为 40）。超出部分的能量会被舍弃。
  - **尺度说明**：Energy 是模型内部数值标尺，不直接等同真实生态学单位。UI 中建议以归一化值（`Energy / maxEnergy`）观察状态变化。

## 2. 能量摄入与消耗（代谢）

每过 1 个 Tick，每个植物细胞都要计算其能量的收入与支出：

1. **光合作用收入（Income）**
   收入与当前的阳光强度和细胞自身的基因相关。高基因（激进）的细胞光合作用效率远超低基因（保守）细胞。
   - `Income = Sunlight * (0.02 + Gene * 0.064)`
   - *（注：目前版本暂时移除了地形肥力设定，所有格子均能公平获取光照。）*

2. **基础代谢支出（Cost）**
   维持生命的基础开销，基因越激进，代谢支出越高（呈平方级增长）。
   - `Cost = 0.001 + (Gene * Gene * 0.006)`

3. **能量扩散（Diffusion）**

   能量扩散是“守恒的局部流动”，但不是无脑平均。想要发生能量外流，必须**同时满足三个条件**：
   1. **自身有余粮**：自身的能量必须 > 0（负能量不会向外扩散）。
   2. **有活体邻居**：8 邻域内至少有一个“活体植物”（空地和墙体不接收扩散）。
   3. **梯度够大**：自身能量必须比所有活体邻居的“平均能量”至少高出 **1.0**（`diffuseGradientThreshold`）。

   如果满足以上条件，细胞才会开始向外“输血”：
   - **外流上限**：每次 Tick 最多只外流自身总能量的 **8%**（由 `diffuseNeighbor=0.08` 与 `diffuseSelf=0.92` 决定），避免“一次性倒空”。
   - **平滑放量**：自身能量高出邻居平均越多，外流量越接近 8% 的上限。如果高出的差值达到 **9.0**（`Threshold + Scale = 1.0 + 8.0`），则满负荷输出 8%。
   - **均分流向**：决定外流的这部分能量，会被平分给周围所有的“活体植物”邻居，一滴不漏。自身保留未外流的所有能量。

   > **调参直觉**：
   > - 想让群落互相“输血”更强（容易拉平差异）：增大扩散上限（`diffuseNeighbor`）或减小启动阈值（`diffuseGradientThreshold`）。
   > - 想让能量更“黏”（内部保持贫富差距）：减小上限或增大启动阈值。

## 3. 密度压力与能量惩罚（拥挤与孤独机制）

Bio-Grid 采用**能量耗尽才凋亡**的抗压机制。不论是过度拥挤还是过度孤独，都会增加能量消耗。

1. **局部拥挤（微观）**
如果一个细胞周围 8 个邻居中，“活体植物”的数量**大于 4 个**（即有 5 到 8 个活体植物邻居时），该细胞会受到局部拥挤惩罚。由于自然界中的资源竞争（如遮光、抢水）往往是非线性的，因此超出 4 个后的密度惩罚会**加速增长**。
- 惩罚计算：`LocalCrowd = Max(0, 邻居数 - 4)`
- 局部能量流失：`CrowdFactor(LocalCrowd) * 0.0018`
  - 5 个邻居（超1）：流失 `1 * 0.0018 = 0.0018`（可感知的额外压力）
  - 6 个邻居（超2）：流失 `2 * 0.0018 = 0.0036`（强压）
  - 7 个邻居（超3）：流失 `6 * 0.0018 = 0.0108`（致死级压力）
  - 8 个邻居（超4）：流失 `15 * 0.0018 = 0.027`（极端高压）

2. **孤独惩罚（微观）**
如果一个细胞周围 8 个邻居中，“活体植物”的数量**小于 2 个**（即只有 0 或 1 个活体植物邻居时），该细胞会因为缺乏群落的微气候保护而受到孤独惩罚。
- 孤独能量流失：`0.016 * NeighborFactor * GeneFactor`
  - `NeighborFactor = (邻居数为 0 ? 2 : 1)`（0 个邻居翻倍）
  - `GeneFactor = 0.4 + Gene * 1.2`（低基因更耐孤独，高基因更吃亏）

其中“活体植物”在实现中指 `Type = PLANT` 且 `Biomass > 0`。

> **结论**：高密度或孤独都不会导致细胞立刻“爆体而亡”。而是会让细胞的能量消耗增加。只有当这些惩罚彻底把细胞的能量（Energy）抽干后，才会触发下文的凋亡逻辑。

## 4. 生长与凋亡（Biomass 结算）

在结算完所有能量的收入、基础开销与密度惩罚后，会得出细胞最终的剩余能量。

- **生长**：如果最终 `Energy > 0`，细胞处于顺境，其生物量（Biomass）会增加 `0.005`，最高不超过 `1.0`。
- **凋亡**：如果最终 `Energy <= 0`，细胞不仅能量归零，还会开始消耗“肉体”来抵债，其生物量（Biomass）会扣除 `0.002`（这给细胞留出了等待日出或熬过干旱的缓冲时间，而不是瞬间暴毙）。
- **死亡判定**：如果由于持续的能量枯竭，导致 `Biomass <= 0`，则该细胞彻底死亡，状态变为 `EMPTY`，基因与能量清零。

## 5. 繁衍与突变（双亲繁殖）

想要繁衍必须同时满足五个苛刻的条件：
1. **成熟度**：自身的生物量（Biomass）必须 > 0.60。
2. **能量储备**：自身的能量（Energy）必须 > 10。
3. **周围有空地**：8 邻域内至少有一个空地（EMPTY）。
4. **不拥挤**：自身的 8 邻域内植物邻居数量必须 <= 4。
5. **有合格的伴侣**：目标空地的周围，除了自身之外，必须还有至少另一个植物邻居同样满足上述的“成熟度”与“能量储备”条件。

如果满足以上条件，细胞会进行分裂：
- **产生后代**：随机选中满足上述条件的一个空地，生成新的细胞（也就是说，这个空地是由两个合格的母体共同孕育的）。
- **初始数值**：后代诞生时的初始生物量为 0.32。
- **能量平分**：两个母体将当前能量的四分之一（25%）分给后代，自己保留剩下四分之三。
- **基因突变**：后代会随机继承两个母体其中之一的基因，但会带有 `±0.04` 的随机突变波动。

## 6. 特殊网格：墙体（WALL）
- 墙体无法生长、死亡或繁衍。
- 墙体能量永久为 0。
- 墙体绝对**不参与**邻居的能量扩散与平均值计算（阻断能量流动）。
- 玩家可用鼠标中键手动绘制墙体，从而人为构造出“生态隔离舱”。
